# Yoast SEO Fields — PWC City Hub Pages

SEO meta is **not** embedded in the GitPress HTML fragments (per `GITPRESS_WEBSITE_STRUCTURE.md`
§12). Set these in WordPress (Yoast / Rank Math) for each page after it is created.

Focus keyphrase, meta title, and meta description below are taken from each city's source
`.docx` ("SEO Setup" + "Target Keywords").

> Note: several drafted meta descriptions run 195–230 characters. Google typically displays
> ~155–160 characters, so Yoast will flag these. Copy is included verbatim from the source docs;
> trim to ~155 chars in Yoast if you want the green bar.
